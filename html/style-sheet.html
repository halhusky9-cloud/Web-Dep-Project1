Style Sheet

Here are the main design choices used for this website’s look and feel.

1. Overall Page Style

The body of the website uses a soft background color to keep the page easy on the eyes. A sans-serif font is used for most text to improve readability. The default text color is a dark gray instead of pure black to create a softer look.

2. Header Section

The header is styled with a bright accent color to make it stand out at the top of the page. The text is centered and uses a different font family (serif) to create contrast from the rest of the page. Padding is added to create space and make the section feel more open.

3. Main Content Section

The main content is placed inside a styled container (div) with a white background. This helps separate the content from the page background. It includes padding for spacing and rounded corners to give the design a modern feel.

4. Footer Section

The footer uses a dark background color with light text to clearly separate it from the rest of the page. The text is centered and spaced evenly using padding.

5. Text Highlighting

A highlight style is used to emphasize important words or phrases. This style changes the font color and makes the text bold to draw attention.

6. Layout Structure (Div Usage)

The page is organized into three main sections using divs:

  • A header div for the title and introduction
  • A main content div for information and sections
  • A footer div for closing information
